Página Principal

Bienvenido a la página principal. Aquí puedes encontrar enlaces a otras páginas que contienen diversos contenidos interactivos.

Página 2: Galería de Imágenes

En la Página 2 encontrarás una galería de imágenes. Cada imagen tiene una animación única que las hace moverse o cambiar de forma, proporcionando una experiencia visual dinámica.

Página 3: Números Aleatorios

La Página 3 genera una tabla de números aleatorios. Es una página interactiva que muestra cómo los números cambian dinámicamente al ser generados aleatoriamente.

Página 4: Enlace Externo

La Página 4 contiene un enlace a una página web externa. Además, en esta página podrás disfrutar de texto e imágenes en movimiento, combinados con una interfaz dinámica e interactiva.

Página 5: Página Vacía

La Página 5 es una página vacía, un espacio para agregar contenido adicional en el futuro. Actualmente, no contiene ningún contenido específico.
